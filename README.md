# 🤖 Project: เจ้าสัว (RobotBrain) 🚀

หุ่นยนต์อัจฉริยะระบบ Offline ที่ใช้ AI ประมวลผลในเครื่อง 100% รองรับการสั่งงานด้วยเสียงภาษาไทยและควบคุมฮาร์ดแวร์ผ่าน Serial (Arduino).

## 🌟 ฟีเจอร์เด่น
- **👂 หู AI (Whisper):** แปลงเสียงพูดภาษาไทยเป็นข้อความได้อย่างแม่นยำด้วย GPU (RTX 4060).
- **🧠 สมอง AI (Gemma 2):** คุยโต้ตอบและคิดวิเคราะห์คำสั่งผ่าน Ollama.
- **🗣️ ปาก AI (TTS):** พูดภาษาไทยได้แบบ Offline (ผ่านการปลดล็อก OneCore Voice).
- **⚙️ ควบคุมหุ่นยนต์:** แยกแยะคำสั่ง "เดินหน้า", "ถอยหลัง", "หยุด" และส่งสัญญาณไปที่บอร์ดจริง.

---

## 🛠️ ความต้องการของระบบ (Hardware & Software)
- **OS:** Windows 10/11
- **GPU:** NVIDIA RTX 4060 (หรือรุ่นที่รองรับ CUDA)
- **Python:** 3.12+
- **Ollama:** ติดตั้งโมเดล `gemma2:2b`
- **Voice:** ติดตั้งภาษาไทยใน Windows Settings

---

## 🚀 วิธีติดตั้ง (Setup)

### 1. ปลดล็อกเสียงภาษาไทย (ครั้งแรกเท่านั้น)
เพื่อให้ Python มองเห็นเสียงไทยของ Windows ให้รันไฟล์นี้ด้วยสิทธิ์ **Administrator**:
```powershell
python fix_voice.py
```

### 2. ติดตั้ง Library ที่จำเป็น
```powershell
pip install -r requirements.txt
# หรือติดตั้งทีละตัว:
pip install openai-whisper speechrecognition pyttsx3 pygame ollama torch win32com
```

---

## 🎮 วิธีใช้งาน (Usage)

### การรันโปรแกรม
```powershell
python RobotBrain.py
```

### โหมดการทำงาน (ตั้งค่าใน RobotBrain.py)
- `USE_MIC = True`: สั่งงานด้วยเสียงผ่านไมโครโฟน
- `USE_MIC = False`: พิมพ์คำสั่งผ่านคีย์บอร์ด (สำหรับเทสระบบ)

### คำสั่งควบคุมหุ่นยนต์
- **"เดินหน้า" / "ไปหน้า":** หุ่นยนต์จะเคลื่อนที่ไปข้างหน้า
- **"ถอยหลัง":** หุ่นยนต์จะถอยหลัง
- **"หยุด":** หยุดการทำงานทั้งหมด
- **เรื่องอื่นๆ:** คุยเล่นกับ AI ได้ตามปกติ (ภาษาไทย)

---

## 📂 โครงสร้างไฟล์
- `RobotBrain.py`: ไฟล์หลักของระบบ
- `fix_voice.py`: สคริปต์ปลดล็อกเสียงภาษาไทย
- `list_voices.py`: ใช้ตรวจสอบรายชื่อเสียงในเครื่อง
- `ears.py`: ระบบทดสอบการฟังเสียงแยกส่วน
